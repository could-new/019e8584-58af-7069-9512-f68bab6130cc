# FrancisMessenger

**FrancisMessenger** est une application Flutter reprenant les codes des réseaux sociaux (type Facebook Mobile). L'application est conçue pour fonctionner avec des données locales, offrant une expérience fluide même sans connexion internet (hors ligne).

## Fonctionnalités

- **Fil d'actualité (Feed)** : Consultez les dernières publications de vos amis.
- **Navigation intuitive** : Barre de navigation inférieure pour accéder à l'accueil, aux amis, aux notifications et au menu.
- **Création de contenu** : Une zone dédiée en haut du fil d'actualité pour partager vos pensées.
- **Hors-ligne par défaut** : Fonctionne avec des données factices locales en attendant une potentielle synchronisation future.
- **Design Responsive** : L'interface s'adapte aussi bien aux écrans de smartphones qu'aux tablettes ou aux navigateurs web.

## Stack Technique

- **Framework** : Flutter
- **Langage** : Dart
- **Stockage** : Mémoire locale (état en mémoire)
- **Design** : Material Design (avec une esthétique inspirée des réseaux sociaux classiques)

## Lancement

Pour lancer l'application sur votre environnement de développement :

```bash
flutter run
```
